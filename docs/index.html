Authentication

plex_auth

Security Scheme Type OAuth2
implicit OAuth Flow
Authorization URL: http://api.peud.io/oauth/dialog
Scopes:
  • read:users -

    read users info

  • write:users -

    modify or remove users

api_key

Security Scheme Type API Key
Header parameter name: api_key

User operations

Retrieve a list of users for a service

Get a list of users from a service

Authorizations:
plex_auth (read:users) api_key
path Parameters
service
required
string
Enum: "peud" "plex" "tautulli" "ombi" "organizr"

Name of service to query

query Parameters
id
integer

Filter users by Plex ID

uuid
string

Filter users by Plex UUID

username
string

Filter users by Plex username

email
string

Filter users by Plex email address

home
boolean

Filter users by Plex home membership

status
string

Filter users by Plex friendship status

admin
boolean

Filter users by Plex admin status

protected
boolean

Filter users by Plex protected status

Responses

200

A JSON array containing user information

403

Forbidden

404

User not found
